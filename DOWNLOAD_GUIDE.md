## 📋 File Formats & Conversion

### HTML Presentation
- **Current Format:** PRESENTATION.html
- **To Convert to PDF:**
  - Browser: Print → Save as PDF (Ctrl+P or Cmd+P)
  - Command: `wkhtmltopdf PRESENTATION.html PRESENTATION.pdf`
  - Online: Use https://cloudconvert.com/html-to-pdf

### Markdown Documents
